# 金融知识入门

用大白话把常见金融概念和产品说清楚，方便零基础读者建立基本框架。

---

## 一、基础概念

**货币**  
就是大家公认的「可换东西的凭证」——现金、银行存款都算。**一句话**：钱是用来交换和储存价值的媒介。

**利率**  
借出钱的人向借钱的人收的「使用费」比例。存款利率 = 银行付给你的利息比例；贷款利率 = 你借钱要付给银行的比例。**一句话**：利率就是「用钱的成本」或「存钱的收益」有多高。

**通胀（通货膨胀）**  
同样一笔钱，能买到的东西变少了——东西在涨价。**一句话**：通胀 = 钱变「毛」了，购买力下降；长期看现金会缩水，所以很多人会拿一部分钱去理财或投资。

**复利**  
利滚利：第一年的利息第二年也算进本金里继续生息。**一句话**：复利 = 利息再生利息，时间越长效果越明显；存钱、理财、贷款里都可能遇到。

## 二、常见术语与产品类型

| 词 | 通俗理解 |
|----|----------|
| **存款** | 钱放在银行，银行付你利息，一般保本、流动性好；活期随时取，定期锁一段时间利息更高。 |
| **银行理财** | 银行代销或自己发的理财产品，收益和风险比存款高，多数不保本；有期限，未到期可能不能随便取。 |
| **基金** | 大家把钱凑一起，交给专业机构（基金公司）去投资股票、债券等；你买的是「份额」，涨跌共担。 |
| **股票** | 买上市公司的「一股」，公司赚钱可能分红，股价涨了你卖出能赚差价，跌了会亏本。 |
| **保险** | 付保费，出险（生病、意外等）时保险公司按合同赔钱；重在保障和转移风险，不是 primarily 用来「赚钱」的。 |

**一句话区分**：存款最稳、流动性好；银行理财收益和风险比存款高一点；基金、股票可能赚也可能亏，风险更大；保险主要是「保事」不是「发财」。

---

## 三、常见产品类型怎么选（思路）

- **要稳、要随时能用** → 优先存款、货币基金等。
- **能接受一点波动、想比存款多赚一点** → 可了解银行理财、债券基金等，看清风险等级和说明书。
- **能接受较大波动、长期不用这笔钱** → 可了解股票、股票型基金等，但要有「可能亏损」的心理准备。
- **要保障（生病、意外等）** → 看保险，先搞清自己缺什么保障、条款里赔什么不赔什么。

**一句话**：没有「稳赚不赔」的，收益和风险往往成正比；按自己的承受能力和用途选，不懂就问清楚再下手。
