## Why

项目已引入 OpenSpec 作为 AI 辅助开发的变更管理流程，但 `note/AI/openspec.md` 目前为空。新成员或首次使用者缺乏系统化的使用指南，难以快速理解安装、目录结构、常用命令及其效果。一份生动全面、易于查阅的文档能显著降低上手门槛，提升协作效率。

## What Changes

- 在 `note/AI/openspec.md` 中编写完整的 OpenSpec 使用文档
- 文档涵盖：安装方式、目录结构介绍、常用命令（propose、apply、explore、archive）、各命令执行后的效果说明
- 语言风格：生动形象、系统全面，便于理解和查阅

## Capabilities

### New Capabilities

- `openspec-usage-docs`：在 `note/AI/openspec.md` 中提供 OpenSpec 的完整使用说明，包括安装、目录介绍、常用命令（propose、apply、explore、archive）及命令效果

### Modified Capabilities

- （无）

## Impact

- **新增文件**：`note/AI/openspec.md` 从空文件变为完整的使用文档
- **无代码变更**：纯文档内容，不影响现有功能
