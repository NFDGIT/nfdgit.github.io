## Context

当前 12 张大卡片每张约 200px 高，3 列布局下需要 4 行 = 800px+，加上间距和 header 远超一屏。需要压缩到一屏内可见全部教程。

## Goals / Non-Goals

**Goals:**
- 一屏内看到全部 12 篇教程
- 按主题分类，结构清晰
- 每个教程保留标题、简短描述和标签
- 主教程突出标记为推荐起点
- 保持 glass 风格、主题支持

**Non-Goals:**
- 不改变教程页面内容
- 不改变导航链接

## Decisions

### 分类方案

| 分类 | 教程 | 定位 |
|------|------|------|
| **基础入门** | 主教程、Tools、Memory、Prompt | 零基础到能跑 |
| **架构进阶** | Multi-Agent、Skills、RAG、Streaming | 能力扩展 |
| **框架生态** | MCP、OpenClaw | 标准协议和框架 |
| **生产上线** | Deploy、Security | 上线准备 |

### 布局设计

每个分类是一个 `glass-panel` 区块，内部是紧凑的列表行：

```
┌─ 基础入门 ──────────────────────────────────────┐
│ 🤖 从零搭建 AI Agent  ⭐推荐起点  Python OpenAI │
│ 🔧 添加更多工具        多工具 并行调用           │
│ 🧠 加入记忆            持久化 Token              │
│ ✍️ Prompt Engineering  提示词 Few-shot           │
├─ 架构进阶 ──────────────────────────────────────┤
│ 👥 多 Agent 协作        Orchestrator             │
│ 📋 Agent Skills        SKILL.md 可复用           │
│ 📚 RAG 检索增强        知识库 向量检索           │
│ ⌨️ 流式输出            Streaming 实时            │
├─ 框架生态 ──────────────────────────────────────┤
│ 🔌 试试 MCP            标准协议                  │
│ 🐾 OpenClaw 框架       Skill 系统                │
├─ 生产上线 ──────────────────────────────────────┤
│ 🚀 部署上线            FastAPI 机器人            │
│ 🛡️ 安全与防护          防注入 沙箱               │
└─────────────────────────────────────────────────┘
```

每行：左侧图标+标题（链接），右侧标签 pills。hover 时整行高亮。高度约 44px/行，4 组共约 600px，一屏可见。

## Risks / Trade-offs

- **[信息密度增加]** → 描述从长文本缩为标签式，足够区分各教程
- **[移动端]** → 标签可隐藏或折行，标题始终可见
