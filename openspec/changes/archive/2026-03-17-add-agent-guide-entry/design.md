## Context

首页使用 `home-grid` 卡片网格展示站点入口，当前有 6 张卡片（笔记、小游戏、相册、小工具、保险、组件库），使用 `anim-slide-up` + `stagger-1` 到 `stagger-6` 实现错开入场动画。Agent 教程页面已在 `guide/agent/index.html` 完成。

## Goals / Non-Goals

**Goals:**
- 首页新增第 7 张卡片链接到 Agent 教程
- 卡片风格与现有一致（图标 + 标题 + 描述）
- 入场动画保持错开节奏

**Non-Goals:**
- 不修改教程页面本身
- 不调整现有卡片顺序

## Decisions

### 决策 1：卡片位置

**选择**：放在现有 6 张卡片之后（第 7 张），使用 `stagger-7`。

**理由**：教程是新增内容类型，放最后不打乱现有用户认知。需在 `base.css` 补充 `stagger-7` 类。

### 决策 2：卡片图标

**选择**：使用 `🤖` 机器人 emoji。

**理由**：直观表达 AI Agent 概念，与现有卡片图标风格一致。

## Risks / Trade-offs

- **[7 张卡片网格布局]** → 现有 3 列网格下第三行只有 1 张卡片，视觉上可能略显不对称，但 `auto-fill` 自适应布局可以接受。
