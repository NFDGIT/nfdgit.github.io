## Why

已创建的 Agent 快速入门教程页面（`guide/agent/index.html`）没有从站点任何位置链接进入，用户无法发现它。需要在首页添加入口卡片。

## What Changes

- 在首页 `index.html` 的卡片网格中添加 Agent 教程入口卡片
- 更新首页副标题，加入"教程"分类
- 在 `base.css` 中补充 `stagger-7` 动画延迟类（当前只到 stagger-6）

## Capabilities

### New Capabilities

无新能力，仅 UI 导航变更。

### Modified Capabilities

无规格级别变更，纯展示层调整。

## Impact

- `index.html`: 添加一个卡片元素
- `assets/css/base.css`: 添加 stagger-7 动画延迟
