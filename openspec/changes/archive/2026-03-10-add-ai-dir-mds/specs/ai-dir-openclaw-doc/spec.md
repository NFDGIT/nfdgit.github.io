## ADDED Requirements

### Requirement: 安装说明

文档 MUST 包含 OpenClaw 的安装方式说明，包括一键脚本（macOS/Linux/Windows）及 npm 手动安装，以及安装后验证步骤（如 `openclaw doctor`、`openclaw dashboard`）。

#### Scenario: 读者按文档完成安装

- **WHEN** 读者按照文档中的安装步骤操作
- **THEN** 读者能够成功安装 OpenClaw 并通过验证命令确认运行正常

### Requirement: 功能点说明

文档 MUST 介绍 OpenClaw 的核心功能点，至少包括：本地运行、全系统访问、浏览器控制、多平台聊天（Channels）、持久记忆、Skills 与插件、Gateway，并说明各功能的作用。

#### Scenario: 读者理解功能

- **WHEN** 读者阅读功能点说明
- **THEN** 读者能够理解各功能的用途及适用场景

### Requirement: 语言风格

文档 MUST 使用简体中文撰写，语言生动形象、系统全面，便于理解和查阅。

#### Scenario: 读者阅读体验

- **WHEN** 读者阅读文档全文
- **THEN** 读者能够感受到语言友好、结构清晰，且内容覆盖安装与功能等核心主题
