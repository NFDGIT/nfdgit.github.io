## Why

Agent 教程目前覆盖了手写 Agent（纯 OpenAI SDK）和 MCP 协议，但缺少对主流 Agent 框架的介绍。OpenClaw 是 2026 年最热门的开源自主 Agent 框架（247K+ GitHub stars），支持本地部署、50+ 集成、技能市场和 MCP。增加 OpenClaw 教程可以让读者从"手写 Agent"平滑过渡到"生产级 Agent 框架"。

## What Changes

- 新增 `guide/agent/openclaw.html` 教程页面，4 步快速入门：理解 OpenClaw → 安装启动 → 连接消息平台 → 创建自定义 Skill
- 在主教程 Step 6 的"下一步"卡片区域新增 OpenClaw 入口卡片
- 更新所有子页面底部的教程导航链接，将 OpenClaw 加入阅读路径
- 更新主教程阅读顺序提示文字
