# billiards-game（变更增量）

## ADDED Requirements

### Requirement: 3D 模式下瞄准与出杆方向与 2D 一致

在 3D 显示模式下，系统 SHALL 将用户在球桌区域上的拖拽（指针按下与移动）转换为与 2D 模式相同的逻辑坐标；同一屏幕位置（相对于当前可见的球桌视图）在 2D 与 3D 下 SHALL 对应同一逻辑点，从而使瞄准线与击球方向与用户拖拽方向一致，不发生「往左拖却往右打」等方向错乱。

#### Scenario: 3D 下拖拽方向与击球方向一致
- **WHEN** 用户在 3D 模式下在球桌视图上从一点向另一方向拖拽并释放
- **THEN** 系统 SHALL 使击球方向与拖拽方向一致（与 2D 下在同一相对位置、同一拖拽轨迹时的击球方向一致）

#### Scenario: 3D 与 2D 同位置同拖拽同结果
- **WHEN** 用户在 2D 与 3D 模式下分别在「同一相对屏幕位置」做相同拖拽轨迹（例如从球桌中心向右下拖）
- **THEN** 两种模式下得到的击球方向与力度 SHALL 一致（逻辑坐标换算一致）
